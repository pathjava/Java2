Задача 1. Перевести проект в Maven
1.1 Создать на основе кода задания H5.2 данного курса (калькулятор с тестами и приёмом информации через командную строку) maven-проект правильной структуры с тестами. В результате работы фазы verify цикла default должен формироваться jar со всеми необходимыми зависимостями.


Задача 2. Создать плагин Maven
2.1 Написать maven-плагин, отправляющий на заданный email (берётся из секции configuration) отчёт о сформированном jar с метаинформацией по нему и вложением в письмо самого jar-a.
Настройки на SMTP сервер получать из параметров конфигурации данного плагина.
Например:

<configuration>
    <emailTo>emailTo@domen.ru</emailTo>
    <emailFrom>emailFrom@domen.ru</emailFrom>
    <authServ>smtp.domen.ru</authServ>
    <authUser>...</authUser>
    <authPass>...</authPass>
</configuration>

2.2 Внедрить созданный maven-плагин в жизненный цикл сборки java-проекта из задачи 1, привязав его к фазе install цикла сборки (только для maven-проекта задачи 1)
