Задача 1. Профайлер в синглтоне
1.1 Оформить в виде синглтона (для многопоточного приложения) класс профайлер ru.progwards.java1.lessons.datetime.Profiler из начального курса. Все методы и свойства сделать нестатическими.
1.2 Адаптировать профайлер к работе в многопоточном приложении. При этом рекурсия (прямая и косвенная) по-прежнему запрещены.

Задача 2. Абстрактная фабрика в синглтоне
2.1 Создать в виде синглтона (для многопоточного приложения) абстрактную фабрику, которая создаёт объекты ByteInteger, ShortInteger, IntInteger (пакет bigints из начального курса) и возвращает их в виде абстрактного класса AbsInteger. Тип создаваемого объекта определяется исходя из диапазона числа, передаваемого в фабрику.

Задача 3. Proxy в качестве фильтра
3.1 При помощи паттерна проектирования Proxy создайте фильтр "выбросов" датчика GPS. Отбрасываются точки, которые не удовлетворяют правилу "3 sigma". Случайной величиной считать скорость движения объекта. Математическое ожидание и дисперсию считать по рекуррентным формулам.
Первые 50 точек не фильтровать, но использовать для статистики.
Класс датчика GPS:
public class GPS {
public double lat; // широта
public double lon; // долгота
public long time; // время в мс
}
